# Identidad Visual de Kanarii

Antes de crear o modificar CUALQUIER componente visual, lee:

  docs/design-system.md

Este archivo contiene la paleta exacta, tipografía, patrones de
componentes, radios, sombras, animaciones y lista de prohibidos.

## Resumen ejecutivo (no sustituye al documento completo)

- Fondo página: `#FDFBF7` | Modales y tarjetas: `white`
- Fuentes: Besley (serif, títulos) + Outfit (sans, cuerpo)
- Acento: `#A5A58D` normal → `#6B705C` hover
- Focus / cálido: `#CB997E`
- Bordes: `#EAE2D6` | Texto: `#4A4E4D` | Secundario: `#8A817C`
- Modal de referencia: `AdminSolicitudesView.tsx`
- PROHIBIDO: glassmorphism, gradientes en botones,
  bordes laterales de color, neón, sombras de color

## Señal de alerta

Si vas a proponer algo descrito como "premium", "glassmorphism",
"HSL balanceados" o "experiencia inmersiva" → detente y consulta
docs/design-system.md primero.
